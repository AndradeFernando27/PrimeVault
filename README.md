# PrimeVault

## 📋 Descrição

PrimeVault é um projeto de CRUD que simula um gerenciador de contas, permitindo que usuários façam login e acessem suas contas através de uma interface web intuitiva.

O sistema é dividido em duas partes:

* **Backend**: API desenvolvida em .NET.
* **Frontend**: Aplicação web construída com React.js.

---

## 🚀 Como executar o projeto

### ✅ Pré-requisitos

* [.NET 7](https://dotnet.microsoft.com/en-us/download/dotnet/7.0)
* [Node.js (versão LTS recomendada)](https://nodejs.org/)
* [Visual Studio Code](https://code.visualstudio.com/) ou qualquer IDE compatível com C# e JavaScript
* [MySql Worchbench](https://dev.mysql.com/downloads/workbench/)

---

### ⚙️ Passo a passo

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/PrimeVault.git
cd PrimeVault
```

---

### 2. Executando a API (.NET)

No terminal, navegue até a pasta da API:

```bash
cd PrimeVaultApi
```

Em seguida, execute os comandos:

```bash
dotnet build
dotnet run
```

A API estará disponível no endereço:
[http://localhost:5000/](http://localhost:5000/)

---

### 3. Executando o Frontend (React.js)

Abra um **novo terminal** e navegue até a pasta do frontend:

```bash
cd primevault.client
```

Depois, instale as dependências e inicie o projeto:

```bash
npm install
npm run dev
```

O frontend estará disponível em:
[http://localhost:3000/](http://localhost:3000/)

---

## 🔗 Endpoints e acesso

* **API:** [http://localhost:5000/](http://localhost:5000/)
* **Frontend:** [http://localhost:30000/](http://localhost:30000/)

---

## 🛠️ Tecnologias utilizadas

* **Backend**: C#, .NET 7
* **Frontend**: React.js, JavaScript
* **Gerenciamento de Pacotes**: npm
* **IDE Recomendada**: Visual Studio Code

---

## 💡 Observações importantes

* Certifique-se de que as portas `5000` (API) e `30000` (Frontend) estejam livres antes de iniciar a aplicação.
* Use o **Google Chrome** para uma melhor experiência de uso e compatibilidade.
* Caso encontre erros ao rodar a aplicação, verifique se todas as dependências estão instaladas corretamente.
